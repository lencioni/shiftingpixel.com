My father

Caution tape by Joe Lencioni

October 5, 2005 by Joe Lencioni
ƒ/4.0, 1/180 s, 100 ISO

Similar photos

They were putting a path in through Linnaeus Arboretum this summer when I was taking photos and there was this caution tape all over the end of one of the paths. It’s nothing too exciting really because I was going to go shoot some stuff tonight but it started to rain like crazy. Oh well…

Post-processing: desaturate with channel mixer adjustment layer on mostly blue channel, add a little bit of red tinge using channel mixer adjustment layer, add red and green using curves adjustment layer, resize and sharpen luminosity channel in lab mode.
